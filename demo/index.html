Rétouch

A canvas-based image editor you can drop into any web app. Powered by fabric.js.

Try the demo

Live Demo

Drag & Drop

Drop images or click to browse. Supports JPEG, PNG, WebP, and GIF formats.

Crop & Rotate

Freeform or fixed aspect ratio cropping with a precise handle-based overlay.

Adjust

Fine-tune brightness, contrast, and saturation with real-time canvas preview.

Powered by fabric.js

Built on the battle-tested fabric.js canvas engine. Tree-shakeable ESM + CJS builds.

Get started

$ npm install @retouchjs/core

or read the documentation
